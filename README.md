## 🚀 Deployment

This website is deployed on **Cloudflare Pages** and served at:

- **Live URL:** https://alifassociates.com
- **www URL:** https://www.alifassociates.com
- **Cloudflare URL:** https://alifassociates.pages.dev

### How to Update the Website

1. Make your edits to `index.html`
2. Go to this repository on GitHub
3. Click `index.html` → click the **pencil icon ✏️**
4. Paste your updated code
5. Click **"Commit changes"**
6. Cloudflare automatically redeploys within **30 seconds**

No build steps. No command line. No server management needed.
